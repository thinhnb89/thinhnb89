Selenium Grid 允许通过将客户端发送的命令路由到远程浏览器实例来在远程机器上执行 WebDriver 脚本。 

Grid 的目标:

* 提供一种在多台机器上并行运行测试的简单方法
* 允许在不同的浏览器版本上进行测试
* 启用跨平台测试
